tripal_phylotree
================

LIS project- tripal module for chado phylogeny tables.

requirements:

* Tripal2, Drupal7

quickstart:

* Clone into drupal/sites/all/modules/tripal
* drush pm-enable tripal_phylotree
* see Admin | tripal | extension modules | phylotrees | sync tab
